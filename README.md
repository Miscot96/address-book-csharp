# Address Book (C#)

Console Address Book app with search, edit, and JSON persistence in C#

## Features
- Add contacts
- Edit contacts
- Delete contacts
- Search contacts
- View all contacts
- Data persistence using JSON

## Project structure
- `Program.cs` - main program and menu
- `AddressBook.cs` - contact management logic
- `Contact.cs` - contact model

## Technologies
- C#
- .NET
- System.Text.Json

## How to run
1. Open in Visual Studio
2. Run the project
